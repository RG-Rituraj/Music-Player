# Task 4: Interactive Music Player 🎵

A feature-rich, dark-themed modern Music Player web application delivering a seamless media streaming user experience.

## 💡 Project Logic & Explanation
- **Audio API Integration:** Developed by utilizing the native `HTML5 Audio API` in JavaScript to handle standard media play, pause, and tracking states directly through code.
- **Dynamic Progress Bar:** Used the JS `timeupdate` event listener to track the current playback position in real time, updating the timeline slider automatically. Users can also click or drag the slider to seek to any part of the track.
- **Playlist & Metadata Management:** Songs are structured inside a JavaScript Array. When a user clicks `Next` or `Previous`, the index changes, automatically updating the audio source, song title, artist name, and matching album artwork.

## 🚀 Key Features
- Robust media controls (Play, Pause, Next, and Previous).
- Interactive, draggable seek-bar and volume slider controls.
- Dynamic UI updates for song metadata and custom background cover art.
